# Competitive Scan (March 6, 2026)

## Goal
Map current category leaders to MVP scope and define differentiation priorities for pilot users.

- Positioning: AI across CRM + marketing workflows and automation.
- Reference: https://www.hubspot.com/products/artificial-intelligence

2. Jasper Marketing AI (Jasper)
- Positioning: enterprise marketing content operations, campaign and brand workflows.
- Reference: https://www.jasper.ai/solutions/marketing

3. Copy.ai GTM AI Platform (Copy.ai)
- Positioning: go-to-market automation and workflow orchestration.
- Reference: https://www.copy.ai/

4. Salesforce Agentforce + Marketing Cloud (Salesforce)
- Positioning: agentic AI integrated into enterprise CRM stack.
- Reference: https://www.salesforce.com/agentforce/

5. Adobe Experience Platform AI Assistant (Adobe)
- Positioning: enterprise experience and data/insight acceleration.
- Reference: https://experienceleague.adobe.com/en/docs/experience-platform/ai-assistant/home

## What This Means for CMO AI Buddy MVP
- Competing on feature count is unrealistic for a March 16 pilot.
- Winning angle should be speed-to-first-value and decision clarity for smaller teams.
- Pilot should prioritize one high-value loop:
  - ingest context -> generate insight/action plan -> execute one workflow -> measure result

## Differentiation Priorities (Pilot)
1. Time-to-first-value under 5 minutes
- Guided onboarding to first analysis/chat output with one-click next action.

2. Decision-centric dashboard
- Show "what to do next" based on spend, conversion, and funnel bottlenecks.

3. Operator trust layer
- Explicit source traces and confidence labels on analysis outputs.

4. Testable growth loop
- Built-in event funnel and attribution so pilot outcomes are measurable for investor conversations.

## De-scoped for Pilot
- Deep enterprise admin controls.
- Full connector parity across major martech stacks.
- Advanced BI/reporting customization beyond core dashboard and billing/profile essentials.

## KPI Targets for Pilot Wave
- Visitor -> Signup conversion
- Signup -> Verified conversion
- Verified -> First value action conversion
- 7-day return rate for pilot users

## Immediate Product Guidance (March 6-16)
- Keep Stripe in test mode.
- Prioritize auth reliability, onboarding, and funnel instrumentation quality.
- Ship only integrations that are tested in end-to-end happy paths; leave others as clearly labeled beta/demo.

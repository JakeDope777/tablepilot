# Reference Data

## Marketing Metrics

| Metric | Formula | Description |
|--------|---------|-------------|
| CAC | Total Marketing Spend / New Customers | Customer Acquisition Cost |
| LTV | Avg Revenue per Customer x Avg Customer Lifespan | Customer Lifetime Value |
| ROAS | Revenue from Ads / Ad Spend | Return on Ad Spend |
| CTR | Clicks / Impressions x 100 | Click-Through Rate |
| Conversion Rate | Conversions / Total Visitors x 100 | Percentage of visitors who convert |

## Best Practices for Prompt Construction

When generating marketing copy, always include the target audience, desired tone, key messages, and any brand guidelines. For analysis tasks, provide relevant context from memory to improve accuracy and relevance.

## Supported Channels

The system supports email, social media (LinkedIn, Twitter/X), paid search (Google Ads), display advertising (Meta Ads), and content marketing (blog posts). Each channel has specific compliance requirements and best practices stored in the compliance rules.

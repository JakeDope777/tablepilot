# Digital CMO AI - Landing Page

A conversion-optimized landing page for Digital CMO AI built with React 19, Tailwind CSS 4, and Framer Motion.

## Design

**"Editorial Noir"** — Dark luxury tech aesthetic inspired by Linear and Vercel.

- Deep charcoal canvas with electric cyan accents
- Space Grotesk display typography + JetBrains Mono for numerals
- Asymmetric layouts with glow effects and dot-grid textures
- Smooth scroll-triggered animations via Framer Motion

## Sections

1. **Hero** — Value proposition with parallax background and dashboard preview
2. **Social Proof** — Trusted-by logos bar
3. **Features** — Six module cards with generated AI imagery
4. **How It Works** — Four-step process flow
5. **Pricing** — Three tiers (Starter $39/Pro $119/Enterprise Custom) with annual toggle
6. **Testimonials** — Three customer quotes
7. **CTA** — Final conversion section
8. **Footer** — Navigation links and legal

## Development

```bash
cd client
npm install
npm run dev
```

## Build

```bash
npm run build
```

The built files will be in `client/dist/`.
